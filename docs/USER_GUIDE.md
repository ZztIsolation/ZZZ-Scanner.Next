# 使用说明

ZZZ Scanner Next 用来扫描《绝区零》背包里的驱动盘，并导出为 `export.json`。它不会读取游戏内存，也不会注入游戏进程；扫描时只会前置游戏窗口、移动鼠标、点击、滚轮和截图。

## 1. 准备绝区零客户端

请先确认游戏客户端满足这些条件：

- 使用 Windows 版《绝区零》客户端。
- 游戏语言设置为简体中文。
- 游戏字体使用细体。粗体可能影响 OCR 识别。
- 画面比例使用 16:9，例如 `1920x1080`、`1600x900`、`1280x720`。
- 打开背包/仓库里的驱动盘页面，让驱动盘列表和右侧详情面板都能看到。
- 扫描期间不要手动移动鼠标、切换窗口或遮挡游戏窗口。

如果启用了 HDR、显卡滤镜、明显改色的显示效果，可能会影响颜色判断。遇到识别异常时，建议先关闭这些效果再试。

## 2. 启动扫描器

1. 从 GitHub Releases 下载 `ZZZ-Scanner.Next-win-x64-framework-dependent.zip`。
2. 解压到一个普通目录，例如桌面、下载目录或 `D:\Tools\ZZZ-Scanner.Next`。
3. 双击运行 `ZZZ-Scanner.Next.exe`。
4. 弹出管理员权限提示时，点击允许。

管理员权限用于稳定操作游戏窗口。如果系统提示缺少运行时，请安装 .NET 8 Desktop Runtime。

## 3. 开始扫描

1. 点击“检测窗口”，确认扫描器能找到 `ZenlessZoneZero`。
2. 点击“预览详情区”，确认预览图是游戏右侧的驱动盘详情面板。
3. 设置“读取上限”：
   - `0` 表示不限制，尽量扫描全部。
   - 第一次使用建议填 `3` 或 `10`，先做小范围测试。
4. 选择品质：
   - 默认只扫 `S`。
   - 如果也想导出 A 级驱动盘，勾选 `A`。
5. 保持“遇到非15级时停止”开启。这样列表按等级排序时，扫到低等级驱动盘会自动停止。
6. 点击“开始扫描”。

扫描时可按 `Ctrl+Shift+C` 停止。界面里的“高级设置”一般不用改，只有排查问题时再展开。

## 4. 查看导出结果

扫描结束后，点击“打开输出目录”。每次扫描都会生成一个新的目录：

```text
Scans/<扫描时间>/
```

常用文件：

- `export.json`：导出的驱动盘数据。
- `scan.log`：扫描过程日志。
- `ocr_diagnostics.csv`：OCR 性能和耗时记录。
- `*.error.txt`：如果某个驱动盘识别失败，会记录失败原因。

如果结果明显不对，请把这次扫描目录里的 `scan.log`、`export.json` 和错误文件一起发给维护者。不要只发截图，日志更容易定位问题。

## 常见问题

**检测不到窗口怎么办？**

先确认游戏已经启动，并且进程名是默认的 `ZenlessZoneZero`。如果你使用了特殊启动器或进程名不同，可以展开“高级设置”修改进程名。

**预览详情区不是右侧面板怎么办？**

确认游戏是 16:9，且已经打开驱动盘仓库页面。如果仍然不对，说明当前分辨率或 UI 布局需要新的扫描配置。

**扫描过程中想停下来怎么办？**

按 `Ctrl+Shift+C`，或点击“停止”。

**为什么要管理员权限？**

扫描器需要稳定地前置游戏窗口并发送鼠标输入，所以会请求管理员权限。它不会读取内存、注入 DLL 或修改游戏文件。
